## 🚀 ইনস্টলেশন (৫ মিনিট)

### শর্ত:
- ✅ PHP 8.x + MySQL (শেয়ার্ড হোস্টিং OK)
- ✅ cURL এক্সটেনশন
- ✅ PDO এক্সটেনশন

### ধাপ:

1. **ফাইল আপলোড** — সব ফাইল আপনার হোস্টিংয়ে আপলোড করুন

2. **ডাটাবেস তৈরি** — cPanel-এ একটি MySQL ডাটাবেস তৈরি করুন

3. **ইনস্টলার চালান** — ব্রাউজারে যান:
   ```
   https://yourdomain.com/install.php
   ```

4. **তথ্য দিন**:
   - ডাটাবেস হোস্ট, নাম, ইউজার, পাসওয়ার্ড
   - অ্যাডমিন ইউজারনেম ও পাসওয়ার্ড
   - ডেমো ডাটা চাইলে চেক রাখুন

5. **লগইন করুন** — ইনস্টল শেষে:
   ```
   https://yourdomain.com/login.php
   ```

6. **কি কনফিগ করুন** (ঐচ্ছিক, সেটিংস প্যানেল থেকেও পারেন):
   - `config.php` ফাইলে Groq API key দিন
   - WooCommerce API key দিন
   - SMTP সেটআপ করুন (ইমেইল পাঠাতে)

7. **নিরাপত্তা**: `install.php` ডিলিট করুন!

## 🔑 ফ্রি API কি পাওয়ার গাইড

### Groq AI (ফ্রি!):
1. যান: https://console.groq.com
2. সাইন আপ করুন
3. API Keys → Create Key
4. কি কপি করে `config.php`-তে `GROQ_API_KEY`-তে পেস্ট করুন

### WooCommerce REST API:
1. আপনার WooCommerce স্টোরে লগইন করুন
2. Settings → Advanced → REST API
3. Add Key → পারমিশন: Read/Write
4. Consumer Key ও Secret কপি করে `config.php`-তে দিন

## ⏰ ক্রন জব সেটআপ

স্বয়ংক্রিয়ভাবে Price, Inventory, Cart Recovery চালাতে:

### cPanel Cron:
```
0 */6 * * * curl "https://yourdomain.com/api/cron.php?token=YOUR_TOKEN&job=all"
```

### cron-job.org (ফ্রি):
1. https://cron-job.org এ সাইন আপ
2. নতুন জব তৈরি করুন
3. URL: সেটিংস প্যানেল থেকে ক্রন URL কপি করুন
4. প্রতি ৬ ঘন্টা সেট করুন

ক্রন URL সেটিংস প্যানেলে পাবেন।

## 🎮 ডেমো মোড

API কি ছাড়াই অ্যাপ পরীক্ষা করতে:

1. `config.php`-তে `define('DEMO_MODE', true)` সেট করুন
2. বা ইনস্টলারে "ডেমো ডাটা যোগ" চেক রাখুন

ডেমো মোডে:
- ✅ সব এজেন্ট সিম্পুলেটেড উত্তর দেবে
- ✅ ফেক পণ্য, অর্ডার, কার্ট ডাটা দেখাবে
- ✅ লগইন: admin / admin123
- ❌ আসল WooCommerce/Groq কল হবে না
